# rubricas-formativas
Rúbricas Formativas es una plataforma web que permite crear rúbricas, evaluar estudiantes en cada clase y monitorear su progreso durante todo el año escolar. Genera porcentajes de logro, gráficos, retroalimentación inmediata e informes, facilitando una evaluación formativa basada en evidencia y una mejor toma de decisiones pedagógicas.
